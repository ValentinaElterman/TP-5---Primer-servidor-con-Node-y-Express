Instrucciones:
1. instalar Express (npm i express o npm install express)
2. runnear el proyecto (el index - npm run start)
3. control + click derecho sobre el https que aparece (http://localhost:3000) o abrilo manualmente
4. Proba las rutas desde el navegador ingresando a:
- http://localhost:3000/ -> Mensaje de bienvenida.
- http://localhost:3000/hora -> Ver la hora actual.
- http://localhost:3000/fecha-completa -> Ver fecha y hora detallada.

5. Proba las rutas desde postman siguiendo estos pasos:
- Abrir postman y seleccionar iniciar sesión luego.
- Abri el trabajo en visual studio code y runearlo/ejecutarlo (npm run start en la terminal).
- En postman tocas el + y verificar que este puesto el GET y no otra opción, si esta otra opción la cambias a GET.
- Ingresas http://localhost:3000 para ver el mensaje de bienvenida.
- Ingresas http://localhost:3000/hora para ver la hora actual.
- Ingresas http://localhost:3000/fecha-completa para ver la fecha completa.
